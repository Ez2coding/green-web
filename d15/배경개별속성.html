1. background-color속성 : 배경 색상만

p1

2. background-image 속성.
: 배경 이미지 적용하는 속성.
*기본형태 : 가로 x 세로 무한반복해서 이미지 출력함.
{background-image:url(이미지경로);}

p2

3. background-repeat 속성.
: 배경이미지의 반복여부를 설정하는 속성.
1) background-repeat:no-repeat
=> 반복없이 원본 1번만 출력
2) background-repeat:repeat-x
=> x 가로만 반복
3) background-repeat:repeat-y
=> y 가로만 반복

p3

4. background-position 속성.
: 배경이미지가 출력될 위치를 설정 하는 속성.
*기본형태 : 해당태그의 왼쪽,위쪽 (0,0) 기준으로 출력됨.
1) 숫자값으로 표현
2) 방향에 대한 영어키워드로 표현
가로위치 : left / center / right
세로위치 : top / center / bottom
3) 숫자값+키워드 / 키워드+숫자값
4) 숫자값으로 표현시 양수(+) / 음수(-) 둘다 표현가능
양수(+) 작성하면 오른쪽 / 아래쪽 방향
음수(-) 작성하면 왼쪽 / 위쪽 방향.

p4

5. background 속성
: 위에 4가지 속성에 대한 값을 한번에 처리하는 속성.
{background:색상 url() 반복 가로위치 세로위치;} {background:#f00 url(img/logo.jpg) no-repeat -35px bottom;}

p5

6. background-size 속성.
: 배경이미지의 크기를 변경하는 속성.
{background-size:가로크기 세로크기;}
{background-size:30px 50px;}
{background-size:70px auto;}
가로크기 70px, 세로크기는 이미지의 비율에 맞춰 자동으로 지정해서 출력해라.

p6

7. 배경이미지의 스크롤 방법을 지정할때 사용하는 속성.
{background-attachment:fixed;}
해당 태그 영역에서만 배경이미지가 보임.
브라우저 기준으로 배경이미지를 고정시켜주는 형태.

p7

* 여러개의 이미지가 한장으로 되어있는 경우.
1) 출력시킬 배경이미지의 가로x세로 크기를 지정해야함.
2) 배경이미지 속성 적용.
3) 이미지 중 원하는 부분만 보일 수 있도록 배경이미지의 포지션 (위치값)을 변경하여 출력해야함.
